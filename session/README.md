## Descripcion
Este ejercicio implementa un pequeño sistema de login utilizando sesiones en `PHP`. El usuario introduce su nombre y contraseña en un formulario, y el segundo archivo valida las credenciales, comprueba la seguridad de la contraseña mediante expresiones regulares y muestra un mensaje de acceso correcto o error. (Se destruye la sesión)
## Description
This exercise implements a simple login system using `PHP` sessions. The user enters a username and password in a form, and the second PHP file processes the data, validates the credentials, and checks the password using regular expressions. A success or error message is displayed depending on the result. (Session is destroyed afterward)
