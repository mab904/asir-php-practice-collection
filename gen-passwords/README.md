# Generador de Contraseñas Seguras en PHP
Este script genera contraseñas aleatorias y seguras utilizando el estandar usual de contraseñas.
También incorpora una función que mezcla los caracteres mediante el algoritmo [Fisher–Yates](https://es.wikipedia.org/wiki/Algoritmo_de_Fisher-Yates), combinado con random_int(), lo que asegura una aleatoriedad criptográficamente fuerte.
# Secure Password Generator in PHP
This script generates random and secure passwords using typical password standards.
It also includes a function that shuffles the characters using the [Fisher–Yates algorithm](https://en.wikipedia.org/wiki/Fisher%E2%80%93Yates_shuffle), combined with random_int(), providing cryptographically strong randomness.
